Variables

Variables

Variables are alphanumeric and can contain the underscore character (_). It may not start with a number or capital letter. The values can be reassigned later on in the code.

Constants

Variables that begin with a capital letter are constants and cannot be changed

Operators

Precendence

Operators following the following precendance and is read from left to right. Parenthesis change the precendance and go in that order.

  1. Exponentiation
  2. Multiplication
  3. Division
  4. Modulus
  5. Addition
  6. Subtraction
Addition a + b
Subtraction a - b
Multiplication a * b
Division a / b
Exponent a**b
Modulo a % b
Self-Assignment Operator a += y

Assignment

Assignment Operator (=)

x = 10
y = 20
z = 30

Parallel Assignment

x, y, z = 10, 20, 30

Swap Values

x, y = y, x

Strings

Newline within the string

"Text \n"

Variables within a string

Repeating a String

String will be repeated, without spaces, by the number it is multiplied by.

"string" * #

Input

Input with Newline Character

variable = gets

Input without Newline Character

variable = gets.chomp

Integer and Float Input

Gets will get a string. For other data types, use the following to convert to the desired data type.

variable = gets.to_i

Integer gets.to_i
Float gets.to_f
